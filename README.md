PART 3 — How to install it (STEP BY STEP)
✅ Method 1 (RECOMMENDED – safest)

Zip the folder:

vikbooking-paystack.zip

Go to WordPress Admin

Plugins → Add New → Upload Plugin

Upload the ZIP

Activate it


✅ Method 2 (Manual)

Upload the folder to:

wp-content/plugins/vikbooking-paystack/


Then activate from Plugins

AFTER INSTALLATION (VERY IMPORTANT)

Go to VikBooking → Payments

You will now see Paystack

Enable it

Paste your sk_test_… key

Set Test Mode = Yes

Save
